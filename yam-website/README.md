# YAM - Yacht Architectural Management

A professional landing page for yacht project management and owner representation services.

## Project Structure

```
yam-website/
├── index.html          # Main landing page
├── assets/
│   ├── css/
│   │   └── styles.css  # Stylesheet (extracted from HTML)
│   ├── js/
│   │   └── main.js     # JavaScript functionality
│   └── images/         # Directory for images (logo, yacht photos, etc.)
├── README.md           # This file
└── package.json        # Project configuration
```

## Features

- Responsive design optimized for all devices
- Modern CSS with custom properties and animations
- Smooth scrolling navigation
- Contact form with mailto functionality
- Professional yacht industry styling
- Performance optimized with minimal dependencies

## Getting Started

1. Open `index.html` in your web browser
2. For development with live reload, consider using a local server:
   ```bash
   # If you have Python installed
   python -m http.server 8000
   
   # If you have Node.js installed
   npx serve .
   ```

## Customization

- **Colors**: Modify CSS custom properties in `:root` selector
- **Content**: Update text directly in `index.html`
- **Images**: Add yacht photos to `assets/images/` directory
- **Contact**: Update email and phone in contact section

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile responsive design
- Progressive enhancement approach
